# HBTesting

A professional static website showcasing HBTesting's testing consultancy services, with a specialty in AI testing frameworks.

## Overview
This website provides information about our expertise in software testing consultancy, highlighting our focus on AI-driven testing solutions.

## Files
- `index.html`: Main page with company information, services, and AI specialty details
- `style.css`: Professional styling with responsive design
- `script.js`: Basic JavaScript for potential interactivity

## Publishing
To publish this website live:
1. **GitHub Pages**: Enable Pages in your repository settings if hosted on GitHub.
2. **Netlify/Vercel**: Connect your repository or drag-and-drop the files.
3. **Other Hosting**: Upload the HTML, CSS, and JS files to any static hosting provider.

The site is fully static and ready for deployment.
